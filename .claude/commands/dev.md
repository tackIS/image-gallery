# 開発サーバー起動

## 手順

1. ユーザーに起動モードを確認:
   - **フロントエンドのみ**（`npm run dev`）— 高速、UIのみの開発
   - **フルアプリ**（`npm run tauri:dev`）— Tauri含む、バックエンド連携テスト

2. ポート競合チェック:
```bash
lsof -ti:1420 2>/dev/null && echo "Port 1420 in use" || echo "Port 1420 available"
```

3. 競合がある場合、プロセスを終了するか確認してから対処

4. 選択されたモードでサーバーを起動（バックグラウンドで実行）
